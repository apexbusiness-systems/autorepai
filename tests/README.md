# Tests

This directory contains all test files.

## Test Structure

- `unit/` - Unit tests
- `integration/` - Integration tests
- `e2e/` - End-to-end tests
- `fixtures/` - Test fixtures and mock data

## Running Tests

```bash
# Run all tests
npm test

# Run specific test suite
npm test -- tests/unit

# Run with coverage
npm run test:coverage
```
